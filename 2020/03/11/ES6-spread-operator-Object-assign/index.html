Object.assign

  • clone object

    const obj = { a: 1 };
    const copy = Object.assign({}, obj);
    console.log(copy); // { a: 1 }

    let obj1 = { a: 0 , b: { c: 0}};
    let obj2 = Object.assign({}, obj1);
    console.log(JSON.stringify(obj2)); // { a: 0, b: { c: 0}}
    obj1.a = 1;
    console.log(JSON.stringify(obj1)); // { a: 1, b: { c: 0}}
    console.log(JSON.stringify(obj2)); // { a: 0, b: { c: 0}}
    obj2.a = 2;
    console.log(JSON.stringify(obj1)); // { a: 1, b: { c: 0}}
    console.log(JSON.stringify(obj2)); // { a: 2, b: { c: 0}}
    obj2.b.c = 3; // obj1, obj2 모두에 영향을 줌
    console.log(JSON.stringify(obj1)); // { a: 1, b: { c: 3}}
    console.log(JSON.stringify(obj2)); // { a: 2, b: { c: 3}}

    • here thing is c will be affected…

    • but, this

      // 깊은 클론
      obj1 = { a: 0 , b: { c: 0}};
      let obj3 = JSON.parse(JSON.stringify(obj1));
      obj1.a = 4;
      obj1.b.c = 4;
      console.log(JSON.stringify(obj3)); // { a: 0, b: { c: 0}}

    • never affect

  • we could merge object

    const o1 = { a: 1 };
    const o2 = { b: 2 };
    const o3 = { c: 3 };
    const obj = Object.assign(o1, o2, o3);
    console.log(obj); // { a: 1, b: 2, c: 3 }
    console.log(o1); // { a: 1, b: 2, c: 3 }, 대상 객체 자체가 변경됨.

    • need to check first object was changed

  • merge object with other object which has same property

    const o1 = { a: 1, b: 1, c: 1 };
    const o2 = { b: 2, c: 2 };
    const o3 = { c: 3 };
    const obj = Object.assign({}, o1, o2, o3);
    console.log(obj); // { a: 1, b: 2, c: 3 }

    • need to know later property could overwrite

Spread Syntax again

  • we could call function without apply

    • previous way

      function doStuff (x, y, z) { }
      var args = [0, 1, 2];
      // Call the function, passing args
      doStuff.apply(null, args);

    • current way

      doStuff(…args);

    • code is shorter, cleaner and no need to use a useless null!

  • combind array

    arr1.push(…arr2) // arr2의 항목들을 뒤쪽에 추가한다
    arr1.unshift(…arr2) // arr2의 항목들을 앞쪽에 추가한다

    • better way

      var arr1 = [‘two’, ‘three’];
      var arr2 = [‘one’, …arr1, ‘four’, ‘five’];
      // [“one”, “two”, “three”, “four”, “five”]

  • copy array

    • previous way

    • we used slice()

    • current way

      var arr = [1,2,3];
      var arr2 = […arr]; // like arr.slice()
      arr2.push(4)

    • *need to know that object in array is reference. so it’s not copy everything

  • use with Math fn

    let numbers = [9, 4, 7, 1];
    Math.min(…numbers); // 1

    • Math fn is perfect example for spread syntax

  • and it could work with destructuring

    let { x, y, …z } = { x: 1, y: 2, a: 3, b: 4 };
    console.log(x); // 1
    console.log(y); // 2
    console.log(z); // { a: 3, b: 4 }

  • we could remove duplication

    const arr = [7, 3, 1, 3, 3, 7];
    const arr2 = […new Set(arr)] // [ 7, 3, 1 ]

  • combind object

    const person = { name: ‘David Walsh’, gender: ‘Male’ };
    const tools = { computer: ‘Mac’, editor: ‘Atom’ };
    const summary = {…person, …tools};
    /*
    Object {

    "computer": "Mac",
"editor": "Atom",
"gender": "Male",
"name": "David Walsh",

    }
    */

    • if there is same key(property), then last property would overwrite

      const person1 = { name: ‘David Walsh’, age: 33 };
      const person2 = { name: ‘David Walsh Jr.’, role: ‘kid’ };
      const merged = {…person1, …person2}
      /*
      Object {

      "name": "David Walsh Jr.",
"age": 33,
"role": "kid",

      }
      */

    • we could generate same feature with Object.assign
