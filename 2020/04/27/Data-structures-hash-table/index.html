Hash Table

Hash tables are a complex data structure capable of storing large amounts of information and retrieving specific elements efficiently.

This data structure relies on the concept of key/value pairs, where the “key” is a searched string and the “value” is the data paired with that key.

Each searched key is converted from its string form into a numerical value, called a hash, using a pre-defined hash function. This hash then points to a storage bucket – a smaller subgroup within the table. It then searches the bucket for the originally entered key and returns the value associated with that key.

  • Advantages
    • Constant number of operations for each search
    • Constant cost for insertion or deletion operations
